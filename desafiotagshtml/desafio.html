TAGS

  • <!DOCTYPE html>: Declara o tipo de documento e versão HTML.
  • <lang>: Declara qual a linguagem está sendo utilizada nos conteúdos da página, melhora perfomance de SEO
  • <html>: Elemento raiz que envolve todo o conteúdo da página.
  • <head>: Contém metadados, links para estilos e scripts.
  • <meta charset="UTF-8">: Define a codificação de caracteres da página.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configura a visualização em dispositivos móveis.
  • <title>: Define o título da página, exibido na aba do navegador.
  • <body>: Contém o conteúdo visível da página.
  • <header>: Seção de cabeçalho geralmente usada para títulos e navegação.
  • <h1> a <h6>: Definem cabeçalhos de diferentes níveis, sendo <h1> o mais importante.
  • <nav>: Seção de navegação contendo links para outras partes do site.
  • <ul>: Lista não ordenada.
  • <li>: Item de uma lista.
  • <a href="#">: Define um link para outra página ou seção.
  • <section>: Seção de conteúdo com tema ou propósito específico.
  • <p>: Define um parágrafo de texto.
  • <footer>: Seção de rodapé contendo informações de copyright e detalhes finais da página.
